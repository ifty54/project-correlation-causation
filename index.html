def scatter_with_regression(x, y, title, xlabel, ylabel):
    plt.figure(figsize=(8, 6))
    sns.regplot(x=x, y=y, data=df, scatter_kws={'s': 50, 'alpha': 0.6}, line_kws={'color': 'red'})
    plt.title(title, fontsize=14)
    plt.xlabel(xlabel)
    plt.ylabel(ylabel)
    plt.tight_layout()
    plt.show()

Defining a Causal Graph (DAG) & Creating the model

In[16]:
from dowhy import CausalModel
In[17]:
# Create the causal model
model = CausalModel(
    data=df,
    treatment='avg_income',       # what we think is causing the change
    outcome='happyScore',         # what is being affected
    common_causes=['GDP', 'median_income', 'income_inequality']  # potential confounders
)

# Visualize the graph
model.view_model()

Identifying and Estimating the Causal Effect

Causal Estimate: 0.0003362229450547538

In[18]:
# Identify causal effect using backdoor criterion
identified_estimand = model.identify_effect()

# Estimate using linear regression (for simplicity)
estimate = model.estimate_effect(
    identified_estimand,
    method_name="backdoor.linear_regression"
)

print("Causal Estimate:", estimate.value)

Refuting the Estimate

In[]:
# Add placebo test
refutation = model.refute_estimate(
    identified_estimand,
    estimate,
    method_name="placebo_treatment_refuter"
)
print(refutation)

# Add random subset refuter
refutation2 = model.refute_estimate(
    identified_estimand,
    estimate,
    method_name="data_subset_refuter"
)
print(refutation2)

I replaced avg_income with a random variable & the effect dropped to near zero, meaning the original result is real, not random.

Income does not bring a strong effect on happiness in this data.

Just because they are related doesn't mean one causes the other.

This shows the difference between correlation and causation.

Final Result: This project shows that average income has a very small causal effect on happiness. Though the effect is tiny, the model is valid, stable, and statistically reliable.
